void TicTacToe.iniciarJuego ( )
inline

Método que inicializa el tablero y establece el jugador actual.

Este método crea un nuevo tablero de 3x3 y lo llena con el carácter '/' para indicar que todas las celdas están vacías. Además, establece al jugador actual como 'X', indicando que es su turno para comenzar el juego.
